# Tattoo Artist Booking System (MVP)

## 📌 Overview
A minimal-cost, reliable booking system connecting tattoo artists and clients.
This MVP includes booking requests, approvals, and payment proof validation.

## 🛠️ Tech Stack
- Frontend: Next.js (React + TypeScript + Tailwind)
- Backend / Database: Supabase (Postgres + Auth)
- Payments: Stripe (Test Mode)
- Hosting: Vercel
- Version Control: GitHub

## 🚀 Roadmap (MVP)
- Day 1–2: Setup & Foundations
- Day 3–4: Core Features (Booking + Auth)
- Day 5–6: Payments + Sync Flow
- Day 7: Testing & Final Deployment
